The Project

The "7 Wonders" project is a responsive website built using HTML, CSS, and JavaScript to showcase the Seven Wonders of the World. It features a clean UI, mobile-friendly design, and structured content layout, ensuring a seamless experience across devices. This project highlights my skills in front-end development, responsive design, and user interface creation. This project helped strengthen my front-end development skills, particularly in responsive design, layout structuring, and creating interactive web pages. It also reflects my ability to build real-world web applications with a balance of design and functionality.
